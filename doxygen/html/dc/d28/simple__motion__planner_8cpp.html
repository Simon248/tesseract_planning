Detailed Description

The simple planner is meant to be a tool for assigning values to the seed. The planner simply loops over all of the MoveInstructions and then calls the appropriate function from the profile. These functions do not depend on the seed, so this may be used to initialize the seed appropriately using e.g. linear interpolation.
