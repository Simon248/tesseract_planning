\doxysubsection{Detailed Description}
This example demonstrates using trajopt to plan in an \char`\"{}online\char`\"{} manner. As the environment is changed (using the joint state publisher), the system will attempt to avoid collisions with the moving object and follow the moving target. 

Note\+: If the target moves too quickly the solver can get stuck in an infeasible point. That is the nature of how the solver is working. Higher level intelligence, a larger step size, or changing the target to a cost can solve that problem.
