\doxysubsection{Detailed Description}
This is a base class for generating instances of processes as tasks such that they may be executed in parallel. A typical workflow would be task t = task\+\_\+generator.\+generate\+Task(input, taskflow) 

Only unique pointers should be used because of the ability to abort the process. With recent changes this may no longer be valid but need to investigate.
