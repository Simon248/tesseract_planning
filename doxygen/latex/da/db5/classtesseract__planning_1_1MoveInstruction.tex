\doxysubsection{Detailed Description}
The move instruction is used when defining the results of a motion planning request. 

This instruction contains two profiles \textquotesingle{}profile\textquotesingle{} and \textquotesingle{}path\+\_\+profile\textquotesingle{} which are similar to industrial robots termination type and Motion Options.
\begin{DoxyItemize}
\item profile (Termination Type)\+: is used to define a set of costs/constraints associated only with the waypoint assigned to this instruction
\item path\+\_\+profile (Motion Options)\+: is used to define a set of costs/constraints associated only with the path taken to the waypoint assigned to this instruction 
\end{DoxyItemize}
