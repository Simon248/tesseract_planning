\doxysubsection{Detailed Description}
This contains the result for the process planning request. 

Must check the status before access the problem results to know if available. \begin{DoxyNote}{Note}
This stores a shared future and is copy-\/able to allow access from multiple threads 

This must not go out of scope until the process has finished 
\end{DoxyNote}
